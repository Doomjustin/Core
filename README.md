# Nevermore Core
Nevemore Core Components


## How to use with CMake
```cmake
target_link_libraries(<your target> PRIVATE/PUBLIC/INTERFACE SF::Core)
```

## Header Template
```c++
#ifndef NEVERMORE_TEMPLATE_H
#define NEVERMORE_TEMPLATE_H


namespace SF {

} // namespace SF

#endif // NEVERMORE_TEMPLATE_H
```
